# IITBBSRHACKATHON-2026
INTERSTELLAR ASTEROID DETECTOR

INTERSTELLAR ASTEROID DETECTOR is an immersive high-fidelity space exploration dashboard and planetary defense command center. By combining real-time NASA telemetry with a custom-trained Convolutional Neural Network (CNN) the system allows users to monitor Near-Earth Objects (NEOs) and classify deep-space phenomena with the precision of a modern astronomical observatory.
